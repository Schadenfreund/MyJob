# Changelog

All notable changes to this project will be documented in this file.
The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [1.0.0] - 2026-01-11

### Added
- **Portable Data Structure**: Implemented `UserData` folder for portable settings and user records.
- **Anonymization Tools**: Added scripts to generate anonymized test data (including Homer Simpson presets).
- **Release Automation**: Integrated `build-release.ps1` for automated Windows packaging.
- **Bilingual Support**: Initial structure for German and English CV/Cover Letter generation.
- **Documentation**: Added tailored `README.md`.
